Como instalamos Vite:

en la terminal:  npm create vite@latest (para la ultima version)
                 npm create vite@6      (para una version especifica) => en mi caso use la 6.

Como correr un proyecto en vite:

en la carpeta del proyecto en la terminal, poner:

vite => "si no funciona usar:" npm install y luego npm run dev
